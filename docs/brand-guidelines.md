> Last updated: 2026-03-18

## 1. Brand Identity

### Who is Tony Tin Nguyen?

A senior Vietnamese technologist and entrepreneur with 19+ years building products for global brands and fast-growing startups. Currently Head of Tech Partnership at **Ecomdy Media** (TikTok Marketing Partner). Co-founder of **GDG Mien Trung**. Writer at **Coder Horizon** (coderhorizon.com).

### Brand Archetype: The Black Mage

Inspired by Vivi Ornitier (FFIX) — a quiet, deeply powerful mage who wins through intellect, not brute force. Commands magic (code) that reshapes reality. Finds strength in knowledge and curiosity, not combat. The glass cannon: fragile on the surface, devastating when unleashed.

**Three dimensions:**
- **The Engineer** — Mastery over code and systems; MAG is the primary weapon
- **The Explorer** — Curious across stacks, markets, cultures; Danang → Singapore → global
- **The Mentor** — Writes for developers, shares hard-won lessons, community builder

| Token | Hex | Usage |

| `border-subtle` | `#1e293b` | Card/panel borders |

| `text-muted` | `#94A3B8` | Secondary text, timestamps |
| `status-green` | `#4ADE80` | Online indicator, success |
| `mage-purple` | `#bb77ff` | Job title accent (Black Mage), MP bar |

**Accent palette (FFIX status):**
- `#ff6b6b` — Fire (passion, urgency)
- `#a8e6cf` — Regen (growth, learning)
- `#ffe66d` — Haste (momentum, shipping)
- `#dda0dd` — Reflect (strategy, architecture)

### Typography

| Role | Font | Weight | Use |
|---|---|---|---|
| Display / Headings | Cinzel | 600–700 | Hero name, section titles, FFIX panel headers |
| Body | DM Sans | 400–600 | Bio, blog posts, descriptions |
| Mono / Tech | Share Tech Mono | 400 | Code snippets, stat panels, FFIX menus |

**Type scale:**
- Hero: `3xl–4xl` Cinzel Bold
- Section titles: `13px` uppercase, `0.1em` letter-spacing, blue left border
- Body: `16px` DM Sans, `1.6` line-height
- Mono panels: `12px–13px` Share Tech Mono

### Iconography

- Use **React Icons** (io5, fa, si sets) — no emoji as UI icons
- Preferred sets: `IoLogo*` for socials, Lucide/Heroicons for UI actions
- FFIX ornaments: `◆ ▶ ◈` unicode characters for section decoration

### Spacing & Layout

- Max content width: `900px` (desktop left column)
- Section gap: `32px–48px`
- Card padding: `20px`
- Border radius: `xl` (12px) for cards, `full` for avatars/dots
- Z-index scale: panels `10`, navbar `20`, overlays `30`, modals `50`

## 3. Voice & Tone

### Brand Voice

| Dimension | Direction |
|---|---|
| Formality | Professional but approachable — never corporate |
| Language | Technical precision + plain-language explanations |
| Character | Dry wit, occasional FFIX/gaming references |
| Emotion | Confident, curious, generous with knowledge |

### Tone by Context

| Channel | Tone |
|---|---|
| Homepage / Bio | Concise, confident, human — no buzzwords |
| Blog (Coder Horizon) | Deep, opinionated, practical — written for builders |
| Social (Twitter/X) | Direct, punchy, sometimes irreverent |
| GitHub / Commits | Technical, precise, conventional |

### Do / Don't

**Do:**
- Speak from 19+ years of real experience
- Share specific lessons, numbers, frameworks
- Reference the FFIX theme with lightness — it's flavor, not costume
- Write for Vietnamese devs building global products

**Don't:**
- Use jargon without explanation
- Overpromise or use growth-hacker language
- Generic humble-brags ("passionate about technology")
- Break the dark aesthetic with jarring light colors

## 5. FFIX Design System (Homepage-specific)

The homepage uses a **Final Fantasy IX** aesthetic as a signature visual signature — not a gimmick. It reinforces the "Dragon Knight" archetype and creates a memorable, distinctive experience.

### FFIX UI Rules

| Element | Style |
|---|---|
| Panels | `bg: rgba(8,14,40,0.97)`, `border: 1px solid rgba(200,168,0,0.3)`, rounded-xl |
| Panel headers | `font: Share Tech Mono`, `color: #c8a800`, uppercase, `◆` prefix |
| Stat bars | Gold fill on dark track, gradient glow |
| Section titles | `Cinzel` font, gold border-left on dark, cream text |
| Battle menu items | `▶` prefix, green hover, mono font |
| Save Point (footer) | Pulsing blue crystal SVG |
| Moogle | Flying animated SVG, cursor:pointer, triggers encounter |

## 8. Anti-patterns to Avoid

- Light mode as default (dark is the brand)
- Sans-serif headings — Cinzel is the signature
- Generic blue/white portfolio templates
- Bright, startup-pink color palettes
- Verbose bio text (keep bio under 3 sentences)
- Breaking the FFIX immersion with mismatched UI
